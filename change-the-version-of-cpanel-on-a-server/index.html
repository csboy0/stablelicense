cPanel license tutorial
February 10, 2023

To change the version of cPanel on a server, you’ll need to follow these steps:

  • Check current version: Before upgrading, you’ll need to determine the current version of cPanel installed on the server. You can check the version by logging into the cPanel account, and then clicking on the “Statistics” section.
  • Backup the server: Before making any major changes, it’s important to backup all important data, including databases, files, and email accounts. This will ensure that you can recover from any unexpected issues during the upgrade process.
  • Verify server requirements: Before upgrading, make sure the server meets the minimum requirements for the new version of cPanel. The requirements can be found in the cPanel documentation.
  • Upgrade the server: The upgrade process can be done using the cPanel & WHM Update script or by contacting your hosting provider to perform the upgrade for you. If you choose to upgrade using the cPanel & WHM Update script, you’ll need to run the following command as the root user:

/scripts/upcp

Monitor the upgrade process: The upgrade process can take some time, and you should monitor it closely to ensure it completes successfully. If you encounter any errors or issues, you may need to consult the cPanel documentation or seek assistance from your hosting provider.

Verify the upgrade: Once the upgrade is complete, log into the cPanel account to verify that the version has been changed to the desired version and that all features and functions are working as expected.

Note: Before proceeding with the upgrade, it’s always a good idea to consult the cPanel documentation and seek advice from your hosting provider to ensure a smooth and successful upgrade.
